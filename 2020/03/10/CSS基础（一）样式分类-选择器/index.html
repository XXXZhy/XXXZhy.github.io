CSS样式分类

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>CSS基础</title>
        <!--
            尽量少用属性设置CSS样式
            CSS样式分为
                1.行内样式：标签内
                2.内部样式：头部head标签内
                3.外部样式：引用css文件        （常用：内容和表现分离，方便搜索引擎加载）
            元素分为：
                块级元素：独占一行，对宽高的属性值生效；如果不给宽度，块级元素就默认为浏览器的宽度，即就是100%宽；
                行内元素：可以多个标签存在一行，对宽高属性值不生效，完全靠内容撑开宽高！
                        其中还有一种结合两种模式有点的显示模式：
                行内块元素：结合的行内和块级的有点，不仅可以对宽高属性值生效，还可以多个标签存在一行显示；
                行内标签：包含a、span、em、strong、b、i、u、label、br；
                块标签：包含p、div、ul、ol、li、dl、dt、dd、h1~h6、form；
                行内块标签：img,input,textarea
        -->
        <style>
            /**
             * CSS分为：
             *     1.内联式CSS样式：内联式css样式表就是把css代码直接写在现有的HTML标签中
             *     2.嵌入式CSS样式：在HTML文档中的< head >中使用< style >标记，，被称为内嵌式。
             *     3.外部式CSS样式：采用< link >标签。例如：< link href = “a.css” type = “text/css” rel = “stylesheet” >< /link >
             * 
             * css语法：
             * 选择器{
             *     样式属性1:属性值;
             *     样式属性2:属性值;
             * }
             * 
             * 我想把h1标签字体颜色编程红色，字体居中，字体大小为20像素，字体类型为“微软雅黑”
             */
            h1{
                color: red;/*设置字体颜色*/
                text-align: center;/*设置字体居中*/
                font-size: 20px;/*设置字体大小*/
                font-family: "微软雅黑";/*设置字体类型*/
            }
        </style>
    </head>
    <body>
        <h1>今天天气好晴朗</h1>
    </body>
</html>

CSS选择器

注意：id和class属性需要符合命名规范，否则无效。

命名规范：

​ 1.大小写敏感

​ 2.最佳规则（保证浏览器兼容性）：始终使用字母、数字或连字符，并且始终以字母开头

标签选择器

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>变迁</title>
        <!--
            标签选择器，选择HTML相应的标签名字作为选择器 比如 p  h1   span  ...
        -->
        <style>
            body{
                background-color: aquamarine;
            }
            h1{
                color: red;
                font-weight: 900; /*字体粗度100~900*/
                font-family: "宋体";
            }
            p{
                color: blue;
                border: 1px dashed black;/*边框粗细   边框实线虚线:solid实线 dashed dotted点线    虚线 边框颜色*/
            }
            a{
                color: #52A96E;
                text-decoration: none;/*去掉文本下划线*/
            }            
        </style>
    </head>
    <body>
        <h1>山东济南</h1>
        <p>这是一段文本</p>
        <a href="https://www.baidu.com">百度一下</a>
    </body>
</html>

id选择器

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>id选择器</title>
        <!--
            ID选择器：要给添加样式的标签添加一个id属性
                    以#开头
        -->
        <style>
            #menu{
                color: white;
                font-size: 36px;
                background-color: #52A96E;
                width: 300px;
            }
            #test{
                color: red;
            }
        </style>
    </head>
    <body>
        <header>
            <nav id="menu">
                <ul>            <!-- ul不占一行，是块元素 -->    
                    <li>新闻资讯</li>
                    <li>娱乐天地</li>
                    <li>体育竞技</li>
                    <li>文化传媒</li>
                </ul>
            </nav>
        </header>
        <main>
            <div id="test">
                <p>5万把共享雨伞不到一天全部“小时”</p>
                <article>
                    <section>
                        <h2>国内新闻</h2>
                        <p>嫦娥七号顺利发射成功</p>
                    </section>
                    <section>
                        <h2>国际新闻</h2>
                        <p>国际油价大幅度上涨，美瞳价格提高80美分</p>
                    </section>
                </article>
            </div>
        </main>
    </body>
</html>

class选择器

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>类选择器</title>
        <!--
            类选择器：给要添加样式的元素加一个class属性
            在css中，类选择器以“.”开头
            class选择器的值不是唯一的，可以用“空格”隔开，一个标签用多个class属性
        -->
        <style>
            .test1{
                color: blue;
                font-weight: bold;
            }
            .test2{
                background-color: yellowgreen;
            }
        </style>
    </head>
    <body>
        <ul>
            <li class="test1 test2">新闻</li>
            <li>视频</li>
            <li>体育</li>
            <li>娱乐</li>
        </ul>

        <p class="test1">中国突然做的决定</p>
    </body>
</html>

通配符选择器

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>通配符选择器</title>
        <!--
            *代表通配符选择器，代表当前页面所有的标签共用该样式
        -->
        <style>
            *{
                color: red;
                font-size: 16px;
            }
        </style>
    </head>
    <body>
        <p>hhhhhh</p>
        <h1>aaaaaaa</h1>
    </body>
</html>

后代选择器

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>后代选择器(包含选择器)</title>
        <!--
            后代选择器：选择某个元素下一代
            多个选择器之间用空格隔开
        -->
        <style>
            #menu{
                background-color: darkgoldenrod;
            }
            #menu .test1{
                color: seagreen;
            }
        </style>
    </head>
    <body>
        <div id="menu">
            <h1>商品分类</h1>
            <ul  class="test1">
                <li>家用电器</li>
                <li>男女服装</li>
                <li>日用百货</li>
            </ul>
            <ul  class="test1">
                <li>家用电器</li>
                <li>男女服装</li>
                <li>日用百货</li>

选择器分组

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>选择器分组</title>
        <!--
            选择器分组：创建由逗号隔开的多个选择器，可以将样式应用到单个选择器匹配的所有元素中
        -->
        <style>
            header,main{  /*多个标签动用 , 隔开*/
                border: 1px black solid;
                width: 600px;
            }
            header{
                height: 100px;
            }
            header nav ul li{
                float: left;    /*左浮动*/
                list-style-type: none;
                width: 100px;
            }
            .p1{
                color: red;
                background-color: #7FFFD4;
                width: 200px;
        </style>
    </head>
    <body>
        <header>
            <nav>
                <ul>
                    <li>首页</li>
                    <li>C#程序设计</li>
                    <li>SQL</li>
                    <li>HTML5</li>
                </ul>
            </nav>
        </header>
        <main>
            <article>
                <h1>国际新闻</h1>
                <p>美国开始部署萨德系统</p>
            </article>
        </main>
        <div id="test1">
            <p class="p1">山东合并批录取，部分高校预估线出炉</p>
        </div>
        <div id="test2">
            <p class="p1">景区客流直线上升，青岛旅游旺季提前“驾到”</p>
        </div>
    </body>
</html>
